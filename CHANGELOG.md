# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - Initial Release

### Added
- **Core Platform**: Complete Full-Stack Web Application for Civic Complaint Management.
- **Authentication**: Role-based authentication (Citizen and Authority) using Firebase Auth.
- **AI Microservice**: Integration with Google Gemini for automated text categorization, priority assignment, and executive summaries.
- **Computer Vision**: Integration with YOLOv8s-World for zero-shot object detection on uploaded images.
- **Dashboards**: Separate, tailored dashboards for Citizens and Authorities with real-time stats and data visualization.
- **Analytics**: Comprehensive analytics dashboard featuring charts and area-wise distribution metrics.
- **Location Services**: Geolocation integration and reverse geocoding via ArcGIS and Nominatim fallbacks.
- **Multi-Language Support**: i18n support for English, Hindi, Marathi, Tamil, Telugu, and Kannada.
- **Responsive Design**: Full responsive UI with Dark Mode and Glassmorphism aesthetics.
